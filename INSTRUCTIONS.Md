## Instruções

1. Instale [NodeJS](https://nodejs.org/pt)

2. Instale o yarn
    ```bash
    npm install -g yarn 
    ```

3. Clone o repositório
    ```bash
    git clone https://github.com/LeonardoMelloTrindade/front-challenge-spacex.git
    ```

4. Instale as dependências na raiz do projeto
    ```bash
    yarn install
    ```

5. Inicie o projeto
    ```bash
    yarn dev
    ```
