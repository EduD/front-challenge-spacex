# pl-gabrielmasiero

## Project setup

### Install all dependencies
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve || npx vue-cli-service serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
