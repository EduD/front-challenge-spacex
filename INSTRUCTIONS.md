## Getting started

Set VUE_APP_GRAPHQL_HTTP on `.env` with value `https://api.spacex.land/graphql/`

Install dependencies and running development

```bash
❯ yarn
❯ yarn serve
```

Creating build

```bash
❯ yarn build
```
